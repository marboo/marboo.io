自由选择编辑器

支持 Mou / Vim / Emacs / Sublime Text 等编辑器。

详细信息 »

支持Markdown等标记语言

内置支持Markdown、reStructuredText、Org-mode。通过自定义扩展可以支持任意标记语言。

详细信息 »

轻松浏览笔记

三栏式界面，浏览笔记，宛若使用Reeder来阅读RSS文章一样。

详细信息 »

安全 所有内容，你来保管

笔记等文件存储在本地文件(夹)中，离开Marboo，仍然可以继续访问您的所有文件。Marboo只是实时响应您对文件(夹)的操作。

随心同步

支持使用Dropbox或Git来同步文件到不同的PC上。

笔记外观，你来决定

通过修改CSS文件来定制所有笔记的外观

小范围内，共享笔记

把笔记的链接地址告诉你的小伙伴们，同一网段的伙伴就可以点击查看。(想要关闭共享？重启Marboo即可)

笔记、博客，合而为一

用Marboo来管理Jekyll / Octopress / Hexo / Ghost 的博文是再好不过了。笔记？博客？有必要分那么清吗？发布到WordPress / Ghost 也是极好的。

码农福利，代码管理

内置Pygments，所有的代码文件在Marboo里都语法高亮显示。
