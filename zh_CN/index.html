还在为Evernote不支持Markdown，Git，Vim而烦恼吗？

Marboo 是 最好的 平台式个人知识管理软件。
使用Marboo，你可以：记笔记，发文章，写Web页面，管理代码。而这一切，基于如下2个设计理念：
使用MVC，将内容和表现分离，使用Markdown等置标语言书写内容，用css+js写成的各种模板来做排版显示。
遵循UNIX的KISS哲学，根据用户配置，Marboo调用外部软件来完成各部分的工作：

观看演示视频 @vimeo 或 @优酷
Markdown 记笔记(其他格式？)
Vim/Emacs 编辑内容
HTML 输出模板
Shell Scripts 自定义转化脚本
Git 版本控制，内容同步
Jekyll 发布博客
Make/Rake 自动化
命令行 实时反馈命令行的操作
